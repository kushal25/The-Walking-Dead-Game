# The Walking Dead - The Zombie Battlefield

The Walking Dead game is a 2-D game built using green foot tool where you survive by killing the enemies(Zombies). 

PURPOSE OF PROJECT: Implementing Design Patterns and advocating XP / Lean values while building The Walking Dead Game.

VERSION or DATE: 1.0 / 04-09-2016

HOW TO START THIS PROJECT: Download the project and use green foot tool. Import the project and play around.

DESIGN PATTERNS IMPLEMENTED: Command Pattern, State Pattern, Factory Pattern, Visitor Pattern, Builder Pattern, Decorator Pattern, Observer Pattern

GAME FEATURES: Different types of zombies, different levels, powerups, multiple lives, Guns and Bullets.
